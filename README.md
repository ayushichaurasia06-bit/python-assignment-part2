# Python Assignment — Part 2  
## Restaurant Menu & Order Management System

This project implements a Restaurant Order Management System using Python data structures such as lists, dictionaries, and nested combinations of both.



The system simulates real-world restaurant operations including menu management, cart handling, inventory tracking, and sales analysis.

- Calculated total number of items
- Counted available items
- Identified the most expensive item
- Listed items priced under ₹150
